# AI LLM Assistant

A conversational AI assistant built with Python, Streamlit, and OpenAI API.

## Features
- Chat-style interface
- Conversation memory using Streamlit session state
- Reset chat button
- Clean and simple UI

## Tech Stack
- Python
- Streamlit
- OpenAI API
- python-dotenv
